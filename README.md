# Gobs
Lightweight and minimalist observability for distributed systems.

## Logs
- Found in `logs.go`
- Currently outputs to stdout

## Metrics
- Found in `metrics.go`
- Currently outputs to stdout
- Supports counters, gauges, and histograms

## Traces
- Found in `traces.go`
- Currently outputs to stdout
- Supports spans and traces
